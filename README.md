Interactive map showing Norwegian ferry routes and highlighting which routes are free.

## Sources

**All ferry routes:** Statens vegvesen Vegkart

  https://autopassferje.no/gratis-ferje/

View the map here:

## Description
This project compares the complete set of Norwegian ferry routes from Vegkart with the list of confirmed free ferry connections published by AutoPASS for ferje.
The result is an interactive visualization where ferry routes are classified as:
  **Free ferry routes**
  **Non-free ferry routes**

## Notes
The free-ferry list may change over time. AutoPASS states that the list is updated as ferry connections are approved as free.
